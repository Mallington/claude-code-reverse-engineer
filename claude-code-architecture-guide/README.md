# Claude Code Architecture Guide

A comprehensive reverse-engineered reference for how Claude Code constructs, manages, and sends context to the Claude API. Based on live traffic captures (claude-trace), local file analysis, and runtime observation of Claude Code v2.1.81 (March 2026).

**Goal:** Provide enough detail that you could recreate the core architecture of Claude Code's context management, skill injection, sub-agent dispatch, and tool system from scratch.
