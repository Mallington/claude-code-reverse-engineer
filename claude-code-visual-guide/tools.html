Optimisation

Deferred Tool Loading

When many MCP tools are configured, Claude Code avoids sending every schema upfront. Instead it lists tool names only and provides a special ToolSearch tool for fetching schemas on demand.

Standard
System prompt + 57 full schemas → Large payload (~15–20K tokens)
Deferred
System prompt + ToolSearch tool + names only → Lean payload
↳ Model calls ToolSearch → Schema loaded → Tool usable
Saves ~10,000–15,000 tokens per turn for large MCP configurations
